Rules

Rules are function that used to validate the input, currently we still have these basic built-in function:

  • required : string     "Field is required / cannot be empty."
  • minLength : object     "Field minimum length allowed."
    • value : number     "Minimum length of value"
    • message : string     "Given error message"
  • maxLength : object     "Field maximum length allowed."`
    • value : number     "Maximum length of value"
    • message : string     "Given error message"
